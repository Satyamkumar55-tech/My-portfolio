# 🚀 3D Developer Portfolio

Welcome to my 3D Developer Portfolio! This interactive web experience showcases my projects, skills, and background using immersive 3D graphics and smooth scroll animations.

🔗 **Live Demo:** [https://my-portfolio-xi-gray-89.vercel.app/](https://my-portfolio-xi-gray-89.vercel.app/)

## ✨ Features
- **Immersive 3D Experience:** Built with Three.js and React Three Fiber to bring the portfolio to life.
- **Scroll Animations:** Powered by GSAP (GreenSock) for high-performance, complex scroll-triggered animations and text effects.
- **Modern Tech Stack:** Fast, responsive, and type-safe development using React, Vite, and TypeScript.
- **Fully Responsive:** Optimized for both desktop and mobile viewing.

## 🛠️ Built With

- **[React 18](https://reactjs.org/)** - UI Framework
- **[Vite](https://vitejs.dev/)** - Next Generation Frontend Tooling
- **[TypeScript](https://www.typescriptlang.org/)** - Static type checking
- **[Three.js](https://threejs.org/) & [React Three Fiber](https://docs.pmnd.rs/react-three-fiber)** - 3D Graphics & Canvas
- **[GSAP](https://gsap.com/)** - Advanced Web Animations

## 🚀 Running Locally

To run this project on your local machine, follow these steps:

### 1. Clone the repository
```bash
git clone https://github.com/Satyamkumar55-tech/My-portfolio.git
cd My-portfolio
```

### 2. Install dependencies
Ensure you have [Node.js](https://nodejs.org/) installed, then run:
```bash
npm install
```

### 3. Start the development server
```bash
npm run dev
```
The local server will start, usually accessible at `http://localhost:5173`.

### 4. Build for Production
To create an optimized production build:
```bash
npm run build
```

## 📝 Note on 3D Assets (Git LFS)
This repository uses Git Large File Storage (LFS) for the 3D model files (`.glb` formats). If you clone the repository and the 3D models do not load or appear corrupted, ensure that you have Git LFS installed on your system before cloning/pulling:
```bash
git lfs install
git lfs pull
```
